# Análisis FODA — GreenSpark

**Equipo:** HackHeroes · **Mención:** ENERGÍA · **Lugar:** Santa Cruz de la Sierra, Bolivia

> FODA específico para Santa Cruz (no genérico). Interno = Fortalezas/Debilidades; Externo = Oportunidades/Amenazas.

---

## Matriz FODA

| | **Positivo** | **Negativo** |
|---|---|---|
| **Interno** | **FORTALEZAS** | **DEBILIDADES** |
| **Externo** | **OPORTUNIDADES** | **AMENAZAS** |

---

## 🟢 Fortalezas (interno)

1. **IA real y demostrable** sin hardware: predicción + optimización + agente, todo por software/API.
2. **Comprador que ya gasta** en el problema (generador urbano paga por desechar y necesita ESG).
3. **Modelo asset-light y escalable** (SaaS): márgenes altos, sin activos físicos pesados.
4. **Efecto red y dataset propietario**: mejora con cada operación; difícil de copiar.
5. **Neutralidad de hardware**: se integra con cualquier planta/biodigestor existente.
6. **Equipo multidisciplinario** (técnico + negocio), alineado a los requisitos del evento.
7. **Narrativa de economía circular** alineada a ODS y a la marca de los clientes.

## 🔴 Debilidades (interno)

1. **Dependencia de coeficientes referenciales** hasta tener datos primarios (riesgo de precisión).
2. **No controlamos la transformación física** (dependemos de plantas socias).
3. **Marca nueva sin track record** ni clientes aún.
4. **Dependencia de una API externa** (DeepSeek) para la capa conversacional.
5. **Equipo pequeño** y recursos limitados post-hackathon.

## 🟡 Oportunidades (externo)

1. **~700–900 t/día de orgánico urbano** sin valorizar en Santa Cruz [ESTIMACIÓN] = mercado enorme.
2. **Presión ESG creciente** sobre empresas e instituciones (demanda de reportes).
3. **Saturación del relleno de Normandía** y costos de disposición al alza.
4. **Precedente local exitoso** (Guabirá vende biomasa al SIN) que valida la tesis.
5. **Mercado de créditos de carbono** como fuente de ingreso futura.
6. **Apoyo institucional posible**: GAMSC, CAINCO, universidades, GDG, cooperación internacional.
7. **Tendencia regional de economía circular** y financiamiento verde.

## ⚫ Amenazas (externo)

1. **Baja cultura de segregación en origen** y alta informalidad.
2. **Inestabilidad económica/cambiaria** en Bolivia (afecta inversión y costos de API en USD).
3. **Vacío o cambios regulatorios** en gestión de residuos.
4. **Posible entrada de competidores** (startups o el propio municipio) si el modelo funciona.
5. **Resistencia al cambio** de generadores acostumbrados a "solo botar".
6. **Dependencia de pocas plantas** receptoras al inicio (cuello de botella de oferta).

---

## Estrategias cruzadas (FODA → acción)

| Cruce | Estrategia |
|---|---|
| **FO** (Fortalezas + Oportunidades) | Usar la IA demostrable + la presión ESG para captar rápido a universidades y cadenas como clientes ancla y casos de éxito. |
| **FA** (Fortalezas + Amenazas) | Apalancar el modelo asset-light y la neutralidad de hardware para sumar plantas socias y no depender de una sola; precios en Bs para mitigar lo cambiario. |
| **DO** (Debilidades + Oportunidades) | Cerrar la brecha de datos con un piloto financiado por el interés ESG/municipal; reentrenar el modelo con datos reales. |
| **DA** (Debilidades + Amenazas) | Onboarding guiado + incentivo económico para superar la baja segregación; fallback del agente para no depender del LLM. |
